# API de Clientes com ASP.NET Core

Projeto desenvolvido em ASP.NET Core Web API e cliente com ASP.NET MVC Core.

## Tecnologias 

 - ASP.NET Core
 - EF Core
 - ASP.NET Identity
 - FluenValidation
 - AutoMapper
 - JSON Web Token
 - Swagger
 - SQL Server

## Como testar ?

Para testar as funcionalidades, você precisará:

 - Visual Studio 2019
 - SDK .Net Core
 - SQL Server
 - Postman
